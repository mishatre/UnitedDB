Можно ли увидеть в понятном для пользователя виде содержимое электронного документа?

В форме просмотра электронного документа отображается соответствующая печатная форма, заполненная на основании данных электронного документа. Такое представление позволяет визуально оценить содержимое электронного документа.
В форме также отображаются установленные для электронного документа ЭЦП, пройденные и планируемые статусы электронного документа.

Важно. В форме просмотра электронных документах, формат которых содержит реквизит "Информационного поле", используемый для передачи дополнительных данных по документу (обычно это служебная информация), будет доступна настройка "Отключить вывод информационного поля", которая позволяет не показывать доп.данные электронного документа.
Значение данной настройки система запоминает автоматически при закрытии формы для каждого пользователя.

Какие действия можно выполнить в форме просмотра электронного документа?
  • В зависимости от настроек соглашения (применяется или нет ЭЦП) и статуса (пройденных этапов) в форме можно выполнить команды по переходу на следующий этап.
  • С помощью команды "Все действия" - "Сохранить электронный документ на диск" в форме электронного документа можно сохранить текущую версию электронного документа вместе с файлами ЭЦП в каталог на диск.
  • После выполнения команды на диск будет выгружен файл электронного документа, а также отдельно сохранены файлы с ЭЦП, если такие были установлены на электронный документ.

Важно. Настройка режима отображения вопроса по сохранению списка установленных ЭЦП производится в "Персональных настройках ЭЦП" персональных настройках программы.

Можно ли отклонить электронный документ, если пользователь не согласен с его содержимым?

Для отклонения электронного документа в форме просмотр электронного документа вызвать команду "Отклонить электронный документ" в группе команд "Все действия" и указать причину отклонения, которая в дальнейшем будет доступна для просмотра другим пользователям.

Важно. Операция отклонения электронного документа является необратимой, дальнейшее использование этого экземпляра будет невозможно, потребуется новая версия электронного документа.

Можно ли распечатать электронный документ для предоставления во внешние инстанции на бумажном носителе?

Распечатать электронный документ можно из формы просмотра содержимого через команду "Все действия" - "Печать документа с электронными подписями".
Электронный документ будет выведен на печать в том виде, который виден на экране с дополнительным листом, где будут перечислены установленные ЭЦП, если такие есть.

Важно. Если электронный документ необходимо предоставить во внешние инстанции в электронном виде, то необходимо воспользоваться командой "Все действия" - "Сохранить электронный документ на диск".

Можно ли перезаполнить учетный документ на основании входящего электронного документа?

Перезаполнение происходит по команде "Перезаполнить документ" для непроведенного документа учетной системы.

Важно. Предыдущее содержимое документа учетной системы будет заменено.

Можно ли "перепривязать" входящий электронный документ к другому документу учетной системы?

В форме входящего электронного документа с помощью команды "Все действия" - "Выбрать документ отражения в учете" можно изменить связь электронного документа с документом учетной системы.
Эта команда не перезаполняет документ учетной системы на основании электронного документа (для этого нужно воспользоваться соответствующей командой в форме электронного документа).

Можно ли в программе при обмене между контрагентами подтвердить отправителю электронного документа, что данные получены?

Если на этапе настройки соглашении прямого обмена между контрагентами было задано требование ответной подписи (установлен признак "Подписывать" на закладке "Входящие документы" напротив нужного вида электронного документа), то в форме просмотра электронного документа можно установить ЭЦП на входящий электронный документ по команде "Подписать и отправить подтверждение" или передать его на подпись ответственному лицу по команде "Утвердить электронный документ".

Важно. При обмене первичными документами в эл.виде формата ФНС, требующего для входящего электронного документа формирование ответного титула, после выполнения команды "Утвердить электронный документ" программа создаст новый электронный документ нужного титула и предложить подписать ЭЦП.
