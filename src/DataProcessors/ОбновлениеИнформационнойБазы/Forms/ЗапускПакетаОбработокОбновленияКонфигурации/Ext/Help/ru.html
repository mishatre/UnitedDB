Обработка «Запуск пакета обработок обновления конфигурации»

Используется в контексте обновления на ред. 10.3

Обработка предназначена для пакетного запуска набора обработок, которые должны запускаться после обновления конфигурации до начала работы. Целесообразно использовать данную обработку в случае, когда последовательно запускать отдельные обработки вручную является неудобным либо неоптимальным.

Список обработок отображается в виде дерева. При пакетном запуске будут выполняться только обработки, отмеченные в списке флажками. Для того чтобы определить, какие обработки необходимо выполнить, нужно найти в дереве узел с номером версии, с которой выполняется обновление, и установить в нем пометку. При этом будут помечены все нижестоящие узлы.

Обработка последовательно запускает на выполнение следующие обработки:

  • Конвертация прав доступа пользователей
  • Подчинение касс и ККМ организациям
  • Заполнение страны Россия
  • Заполнение реквизитов документов ценообразования
  • Обновление документов Корректировка долга
  • Обновление данных документов Прочие затраты 
  • Заполнение новых измерений Контрагент, Организация в регистрах накопления
  • Обновление данных НДС расчеты
  • Заполнение ресурса НДС в регистре Продажи
  • Обновление данных по взаиморасчетам в разрезе документов расчетов
  • Изменение вида склада с розничного на оптовый
  • Заполнение регистра сведений Объекты доступа документов
  • Конвертация справочника Дополнительные печатные формы
  • Обновление номеров документов
  • Заполнение ресурса стоимость без скидок в регистре продажи для услуг
  • Обновление данных регистра НДС Партии запасов
  • Установка соответствия проектов номенклатуре
  • Заполнение типа элемента в регистре «Адресный классификатор»
  • Заполнение нового измерения Организация в регистре взаиморасчеты с подотчетными лицами

Результаты запуска каждой обработки отражаются в журнале регистрации
