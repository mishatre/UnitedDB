Справочник "Профили распределения по периодам"

Справочник содержит информацию о различных профилях, по которым перераспределяются плановые данные во времени, например, для распределения плановых показателей (оборотов) введенных за больший период в показатели (обороты) по более коротким составляющим периодам.

Пример. Распределение годовых оборотов по месяцам с использованием профиля некоторой сезонности.

Каждый профиль имеет собственное наименование и два параметра: «Период» и «Подпериод».

  • "Период" - реквизит, определяющий больший период, показатели которого будут распределяться.
  • "Подпериод" определяет малый (составляющий) подпериод, по которому производится распределение.

Кроме того, в табличной части хранится информация о том, с каким весом входит в больший период каждый подпериод.

  • "Доля подпериода" определяет вес вхождения.
  • "Номер подпериода" - определяет номер составляющего подпериода, для которого указывается вес.

Заполнение производится из формы элемента. Дополнительных средств контроля не содержится.

Последовательность заполнения проста – указываются период и подпериод, затем заполняется таблица весов подпериодов в периоде.
