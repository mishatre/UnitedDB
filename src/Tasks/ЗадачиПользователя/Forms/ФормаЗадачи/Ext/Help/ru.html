Задачи с оповещением. Форма задачи

Создание новой задачи

Для успешного создания новой задачи необходимо:  

1). Выбрать пользователя-исполнителя в поле "Исполнитель:"

2). Сформулировать краткое условие задачи в поле "Задача"

Срок исполнения задачи может быть задан в поле "Исполнить до:", но для создания новой задачи не обязательно должен быть указан, предоставляя пользователю фиксировать создание глобальных бесконечных задач, или задач, срок исполнения которых ещё предстоит уточнить. Независимо от этого может использоваться напоминание по задаче, которое можно установить, отметив флаг "Напомнить:" и указав в соответствующем поле значение времени напоминания. Значение параметра "Автор" при создании задачи будет сформировано автоматически.

Действия над задачами

Все действия в форме задачи могут производиться равноценно всеми пользователями, независимо от их роли в текущей задаче. И автор и исполнитель задачи могут изменить введенные значения полей задачи в форме. Ограничением для существующих задач установлены: возможность изменения условия задачи только её автором и только при условии, что задача не выполнена, и, невозможность включения оповещения для выполненной задачи. Значение "Автор" редактированию не подлежит в любом случае.

Включение и выключение оповещения по активной задаче осуществляется только в текущей форме!
