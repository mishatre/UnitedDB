Справочник «Настройки бизнес-процессов»

Справочник «Настройки бизнес-процессов» служит для хранения информации о параметрах задач для каждой точки маршрута бизнес-процесса.

Обязательным является указание вида бизнес-процесса, для которого будет использоваться данная настройка. После того, как настройка использовалась в каком либо из процессов, вид настройки изменять нельзя.

>Для настройки с видом  бизнес-процесса «Согласование заказа покупателя» обязательным является указание исполнителей задач для каждой точки действия карты маршрута. Для видов  бизнес-процессов «Переоценка товаров в рознице»,«Переоценка товаров в АТТ» исполнители определяются алгоритмически, и в справочнике не указываются.

Для вида   бизнес-процесса «Переоценка товаров в рознице»  необходимо указать настройку для точки вложенного бизнес-процесса «Переоценка товаров в АТТ».

Для каждого вида бизнес-процесса  может быть создано неограниченное количество  настроек.
