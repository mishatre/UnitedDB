Бизнес-процесс «Переоценка товаров в рознице»

Бизнес-процесс «Переоценка товаров в рознице» необходим для проведения переоценки товаров на розничных складах при изменении цен компании.

Процесс переоценки состоит из двух этапов:

  • переоценка товаров на складе АТТ
  • контроль переоценки

Этапы переоценки отражены на карте маршрута бизнес-процесса в виде точек «действия». Бизнес-процесс, следуя по маршруту, автоматически в каждой точке «действия» создает задачи. 

На первом этапе для всех розничных складов, у которых в типе цен розничной торговли выбран тип цен из документа «Установка цен номенклатуры», формируются бизнес-процессы «Переоценка товаров в АТТ». В рамках каждого из этих процессов производится переоценка товаров на соответствующем складе АТТ. По завершению всех  процессов переоценки  создается задача контроля, для выполнения которой предоставляется перечень созданных документов по каждому складу и отчет «Сравнение цен АТТ и цен номенклатуры».

Ввести новый экземпляр бизнес-процесса можно путем ввода на основании документа «Установка цен номенклатуры». Вариант прохождения бизнес-процесса по маршруту выбирается в поле "Настройка"  из справочника "Настройки бизнес-процессов" . В настройке для каждой точки «действия» можно  указать представление, детальное описание, планируемый период выполнения и оповещения задачи. В дальнейшем эти параметры используются при создания задач в точках маршрута. При необходимости некоторые точки можно отключить, что приведет к сокращению стандартной карты маршрута. 

Для получения информации о текущей точке маршрута можно использовать «Карту маршрута», которая вызывается по кнопке  в командной панели формы.
