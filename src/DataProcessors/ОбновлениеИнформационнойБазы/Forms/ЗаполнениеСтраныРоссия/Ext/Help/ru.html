Обработка «Заполнение страны Россия»

Обработка используется в контексте обновления на ред. 10.3

В справочник «Классификатор стран мира» добавлен предопределенный элемент «Россия». Данная обработка осуществляет обновление данных о стране в объектах информационной базы. Обработка может выполняться в несколько этапов, либо все этапы могут быть выполнены в один прием.

Обновить адреса контактной информации

На данном этапе происходит обновление записей регистра сведений «Контактная информация», которые имеют тип «Адрес». Для всех Адресов не из России в «представление» добавляется страна

Корректировка справочника Серии номенклатуры

В справочнике происходит замена страны происхождения Россия на предопределенный элемент «Россия» справочника «Классификатор стран мира»

Корректировка справочника Номенклатура
