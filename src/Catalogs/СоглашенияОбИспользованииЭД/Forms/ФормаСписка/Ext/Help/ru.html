Где задаются параметры обмена электронными документами с участниками?

Все параметры обмена с участниками регулярного обмена, задаются в "Соглашении об использовании электронных документов".
В соглашении указывается способ обмена, идентификатор участника обмена, форматы электронных документов и т.д.

Для выгрузки/загрузки данных из файла по однократной сделке все параметры задаются в момент выполнения самой операции и не требуют предварительного указания.

Для работы по повторяющимся сделкам в программе поддерживаются следующие варианты обмена, настраиваемые на соответствующих закладках формы списка:

  • обмен через оператора электронного документооборота (оператора ЭДО);
  • прямой обмен между контрагентами;
  • обмен между организациями в рамках одной компании;
  • прямой обмен с банками.

Важно. Для прямого обмена с контрагентами необходимо вводить и настраивать "Соглашение об использовании электронных документов" для каждого контрагента в отдельности.
Экземпляр соглашения при обмене через оператора ЭДО - один по каждому из операторов ЭДО, где хранятся настройки для всех контрагентов сразу.

Важно. Соглашение прямого обмена имеет более высокий приоритет по сравнению с Соглашением через оператора ЭДО, т.е. если в программе оформлено два действующих соглашения с одним контрагентом (соглашение прямого обмена и соглашение через оператора ЭДО), то программа выполнит отправку электронного документа по "Способу обмена", заданному в Соглашении прямого обмена, если на закладке "Исходящие документы" у используемого "Вида электронного документа" установлен флаг "Отправлять".

Как ввести новое соглашение для обмена электронными документами через оператора ЭДО?

В общих настройках системы включить возможность применения электронной цифровой подписи (ЭЦП), а затем из списка соглашений об использовании электронных документов на закладке "Через операторов ЭДО" ввести новое соглашение:

  • либо по команде "Создать", используя помощник регистрации нового соглашения;
  • либо по команде "Создать новое (ручной режим)", заполнив все требуемые реквизиты вручную в форме нового элемента.

Важно. Обмен электронными документами возможен только по непомеченным на удаление соглашениям, находящимся в статусе "Действует".

Как ввести новое соглашение для прямого обмена электронными документами между контрагентами?

Новый элемент можно ввести из списка соглашений об использовании электронных документов на закладке "Прямой обмен с контрагентами".

Важно. Обмен электронными документами возможен только по непомеченным на удаление соглашениям, находящимся в статусе "Действует".

Как ввести новое соглашение для прямого обмена электронными документами с банками?

В общих настройках системы включить:
- возможность применения электронной цифровой подписи (ЭЦП);
- использование обмена электронными документами для прямого взаимодействия с банками.
Добавить новый элемент из списка соглашений об использовании электронных документов на закладке "Прямой обмен с банками".

Важно. Обмен электронными документами возможен только по непомеченным на удаление соглашениям, находящимся в статусе "Действует".
