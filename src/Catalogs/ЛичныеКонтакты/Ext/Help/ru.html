Справочник «Личные контакты»

Справочник служит для хранения информации о личных контактах пользователя.

Личные контакты доступны для просмотра и редактирования только ответственному за данное контактное лицо. Ответственный не может быть изменен и назначается системой автоматически как текущий пользователь системы. По кнопке Перейти в командной панели формы, можно перейти к списку электронных писем текущего пользователя, при использовании встроенного почтового клиента.
