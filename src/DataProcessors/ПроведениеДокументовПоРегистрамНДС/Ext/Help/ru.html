Обработка "Проведение документов по регистрам НДС"

Обработка предназначена для проведения документов, введенных в информационную базу, по регистрам учета НДС. При этом, в случае если в организации ведется сложный учет НДС (взведен флаг "Организация осуществляет реализацию без НДС или с НДС 0%" в учетной политике по налоговому учету), заново производится проведение по партиям НДС.

При этом в обработке можно указать период, за который необходимо перепровести документы.
