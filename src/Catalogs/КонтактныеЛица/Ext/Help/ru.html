Справочник «Контактные лица»

Справочник служит для хранения информации о контактных лицах всех контрагентов.

Для каждого контактного лица хранится его контактная информация, на закладке «События» отображается история зарегистрированных событий с данным контактным лицом, где можно ввести новые события.

Контактная информация ведется только для контактных лиц контрагентов. Один из пользователей может назначаться основным менеджером покупателя (контрагента), именно этому пользователю будут производиться напоминания о дне рождения покупателя.

Важно. При работе с системой напоминаний используются регламентные задания. Поэтому при работе в файловом режиме в настройках параметров учета на закладке Обмен данными необходимо указать пользователя для выполнения регламентных заданий в файловом режиме и интервал опроса регламентных заданий.

Информация о том какие контактные лица относятся к какому контрагенту хранится в справочнике "Контактные лица контрагентов". Информация о личных контактах пользователя хранится в справочнике "Личные контакты".
