Обработка «Заполнение реквизитов документов ценообразования»

Обработка используется в контексте обновления на ред. 10.3

Обработка осуществляет заполнение новых реквизитов в документах ценообразования

Заполнение Типов оплат для Видов оплат чека ККМ

Осуществляется заполнение реквизита «Тип оплаты» значением «Наличные» в справочнике «Виды оплат чека ККМ».

Старый реквизит справочника «Вид движения денежных средств» удален.

Назначение скидок по информационным картам

Осуществляется анализ информационных карт с типом «Дисконтная», для которых указан процент скидки. На основании этих данных:

  • Создаются элементы справочника «Виды дисконтных карт»
  • Создаются документы «Установка скидок номенклатуры» для каждого вида дисконтных карт (создается пара документов - оптовая и розничная скидка). Датой начала действия скидки назначается начало текущего года, дата документа устанавливается текущей датой.

Заполнение реквизита Вид операции

Выполняется для документов «Установка скидок номенклатуры», «Переоценка товаров в рознице».

Заполнение реквизита Способ расчета цены

Выполняется заполнение реквизита «Способ расчета цены» для расчетных и динамических цен

  • В справочнике «Типы цен номенклатуры»
  • В документе «Установка цен номенклатуры»
  • В регистре сведений «Цены номенклатуры»
