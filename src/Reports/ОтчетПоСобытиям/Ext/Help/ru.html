Отчет «Отчет по событиям»

Отчет по событиям строится на основе той информации, которая введена в документ «Событие». Отчет может быть сгруппирован по контрагентам или по контактным лицам.

Для каждого документа «Событие» отображаются в расшифровке его основные параметры.
 
По кнопке (+) можно произвести расшифровку и посмотреть полную информацию о сути события.
