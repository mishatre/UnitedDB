Документ «Документ расчетов с контрагентом (ручной учет)»

Вспомогательный документ, который не делает никаких движений, предназначен для ведения ручного учета по расчетным документам. Реквизиты необходимые для заполнения:

  • Организация
  • Контрагент
  • Договор

    • Пример. Данный документ создается автоматически при оформлении документа "Ввод начальных остатков по взаиморасчетам" при вводе начальных остатков сумм взаиморасчетов по договорам с установленным флагом "Вести по документам расчетов с контрагентами".
