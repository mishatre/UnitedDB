Форма «Связанные документы»

В форме отображается список документов, связанных с выбранным документом. Под связанными документами понимаются документы, в реквизитах которых содержится ссылка на искомый документ. 

Вызов данной формы осуществляется из форм соответствующих документов,  с помощью меню «Перейти».

Документ, для которого форма была сформирована, отображается в шапке формы.

 

Над документами из списка можно выполнять следующие действия:

  • открытие формы документа
  • пометка на удаление
  • проведение
  • отмена проведения
  • открытие формы списка выбранного документа с позиционированием на документе
  • формирование отчета «Движения документа по регистрам»
  • просмотр проводок документа по бухгалтерскому и налоговому учету
  • печать документа (используется печатная форма, назначенная по умолчанию)
  • обновление списка документов.
