Документ «Рассылка анкет»

Документ предназначен для массовой рассылки анкет.

В документе необходимо выбрать анкету, которую необходимо массово разослать.

В качестве e-mail отправки необходимо выбрать учетную запись либо просто указать e-mail в виде строки.

В качестве e-mail ответов нужно указать тот адрес, на который анкетируемые должны будут отсылать заполненные анкеты.

В случае, если анкета рассылается для уточнения данных, которые уже находятся в базе (например для проверки данных физического лица), можно установить Флажок "Заполнять анкету данными". В этом случае, в рассылаемые анкеты будет добавлена та информация, которая в данный момент есть в базе.

 

В случае, если анкета предназначена для загрузки в справочник "Физические лица", появляется возможность заполнить получателей анкеты списком работников.

Заполнение осуществляется из подменю "Заполнить" на закладке "Получатели".

Если анкеты необходимо разослать не самим сотрудникам, а руководителям их подразделений, необходимо установить флажок "Рассылать руководителям".

 

Непосредственная отправка анкет производится по кнопке "Отправить".
