Обработка «Загрузка результатов анкетирования»

Обработка предназначена для загрузки информации из результатов анкетирования. Загрузка возможна из почты и каталога.

Загрузка информации из почты

Загрузка информации из писем возможна как при использовании внутреннего клиента, так и внешнего.

Важно! Загрузка из внешнего клиента имеет одну особенность. Для того, что бы можно было заполнить список писем из почтового клиента, необходимо снять признак "Только непрочитанные входящие письма".

Для заполнения списка писем, в которых присутствуют анкеты, необходимо нажать кнопку "Обновить".

При загрузке писем из программы Microsoft Office Outlook 2003, появится диалоговое окно с предупреждением. В диалоговом окне необходимо установить флажок "Разрешить доступ на", и выбрать значение из списка "10 минут".

После заполнения списка необходимо расставить флажки для каждой анкеты.

Флажок "Эл. справочника" означает, что информация из анкеты будет загружена в справочник. Вид справочника определяется в анкете.

Флажок "Опрос" означает, что информация из анкеты будет загружена в документ Опрос.

Прежде чем загружать информацию из анкеты, можно посмотреть, какая информация хранится в анкете. Для этого необходимо выбрать анкету. В списке вложений анкеты можно выбрать вложение, тогда в табличном поле "HTML" можно будет увидеть информацию, хранящуюся в анкете.

Для загрузки информации необходимо нажать кнопку "Загрузить".

Загрузка информации из каталога

Для заполнения списка файлов необходимо выбрать каталог, в котором находятся анкеты и нажать на кнопку "Обновить".

После заполнения списка необходимо расставить флажки для каждой анкеты.

Флажок "Эл. справочника" означает, что информация из анкеты будет загружена в справочник. Вид справочника определяется в анкете.

Флажок "Опрос" означает, что информация из анкеты будет загружена в документ Опрос.

Прежде чем загружать информацию из анкеты, можно посмотреть, какая информация хранится в анкете. Для этого необходимо выбрать анкету. В табличном поле "HTML" можно будет увидеть информацию, хранящуюся в анкете.
