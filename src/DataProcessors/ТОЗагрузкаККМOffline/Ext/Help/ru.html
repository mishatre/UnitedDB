Обработка «Загрузка данных в ККМ, подключаемые в режиме Offline»

Данная обработка обслуживания предназначена для загрузки списка товаров в ККМ, подключенные в режиме Offline. При использовании данной обработки предполагается следующий порядок действий:

  • Редактирование списка выгрузки - необходимо задать список товаров, подлежащих выгрузке в ККМ. В процессе редактирования данного списка каждому товару присваивается идентификатор (PLU), который будет присвоен данному товару на кассе. Для перехода к данному списку необходимо нажать кнопку «Редактировать список выгрузки» верхней командной панели формы обработки.
  • Выбор склада - указание склада необходимо для определения цены товаров. Для указания склада используется поле выбора «Склад», расположенное в верхней части формы обработки.
  • Выбор ККМ - для выбора ККМ, подключенной в режиме Offline, используется поле выбора «ККМ Offline», расположенное в верхней части формы обработки.
  • Выбор типа цен - тип цен используется для определения цены товаров. Для его указания используется поле ввода «Тип цен», расположенное в верхней части формы обработки.
  • Редактирование параметров заполнения - при заполнении таблицы выгружаемых товаров можно указать необходимость выгрузки только тех товаров, которые есть на заданном складе. Для использования этой возможности необходимо использовать флажок «Только имеющиеся на складе», расположенный в верхней части формы обработки.
  • Заполнение таблицы выгружаемых товаров - для выполнения данного действия необходимо нажать кнопку «Заполнить», расположенную в верхней командной панели формы обработки.
  • Выгрузка товаров - для выполнения данного действия необходимо нажать кнопку «Загрузить в ККМ», расположенную в верхней командной панели формы обработки.
