Документ «Установка долей распределения по проектам»

В программе предусмотрена возможность отнесения одной позиции номенклатуры или сумм, указанных в одном документе распределять по нескольким проектам. Для этого в справочник «Виды распределения по проектам» вводится наименование видов распределения, а в документ "Установка долей распределения по проектам" ввести доли распределения по проектам. Документ «Установка долей распределения по проектам» может быть введен на основании записи справочника «Виды распределения по проектам».
 
Виды распределения по проектам и доли распределения можно задать заранее и потом использовать возможные виды распределения при указании проектов в в документах и для позиций номенклатуры. При указании видов распределения используется следующий алгоритм:

  • если для документа указан вид распределения по проекту, то при проведении документа суммы по документу распределяются по проектам в соответствии с этим видом распределения и указанными долями проектов;
  • если для документа не указан вид распределения по проектам, то тогда при проведении документа суммы распределяются в соответствии с видами распределения и долями, указанными для номенклатурных позиций.

Анализ данных в разрезе проектов с учетом распределения позиций номенклатуры по проектам можно производить в отчетах «Продажи» и «Закупки». Для группировки данных по проектам надо выбрать реквизит – «Проект».
