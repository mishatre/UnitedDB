Справочник "Настройки обмена данными с сайтом"

Справочник "Настройки обмена данными с сайтом" предназначен для хранения настроек обменов данными. Возможна настройка обмена через единый файловый ресурс или непосредственно на сайт.

 

Не зависимо от типа взаимодействия с сайтом поддержан следующий функционал:

1. Возможность указания объектов для обмена (обмен каталогом товаров и обмен заказами),

2. Возможность полного обмена или обмена только измененными объектами с момента последнего обмена,

3. Возможность настройки периодического обмена данными,

4. Для обмена данными о товарах можно указать ограничения для товаров поставляемых на сайт,

5. Выгрузка изображения товара на сайт,

6. Определение параметров, подставляемых в заказ, приходящий с сайта,

7. Определение действий с заказом, приходящего с сайта.
